# 🛍️ Ecommerce Responsive Website

A fully responsive ecommerce landing page built with HTML, CSS, and JavaScript. It features a sticky navbar, mobile hamburger menu, trending products grid, client reviews, blog/news updates, and a footer contact section — optimized for mobile, tablet, and desktop screens.
#
## ✨Live Demo
 🔗 https://frontend-ofe-commerce.vercel.app/

## 📌Tech Stack
- HTML5
- CSS3 (Grid & Flexbox)
- Vanilla JavaScript
- Font Awesome Icons

## ☀ Project Structure
```
ECOMMERCE-WEBSITE/

  │   ├── banner-3.png
  │   └── (product & blog images)
  ├── index.html

  └── README.md
```

## 👉Features
- Sticky header that activates on scroll
- Mobile hamburger menu with slide-in navigation
- Hero/banner section with call-to-action button
- Trending products grid with ratings, sale/new/hot tags, and wishlist icon
- Client reviews section
- Latest news/blog cards section
- Footer with contact info, social icons, and quick links

## 🎭Responsive Web Design
- Layout scales across mobile, tablet, and desktop breakpoints (1024px, 890px, 750px, 630px)
- Hero background image stays centered and uncropped on smaller screens
- Navbar collapses into a hamburger menu on mobile
- Nav icons and logo resize for smaller screens
- Product and blog cards reflow into fluid grid columns
- Touch-friendly spacing and typography on smaller devices

## 🚩Getting Started
### Prerequisites
- A modern web browser
- (Optional) VS Code with Live Server extension

### Installation
```bash
git clone https://github.com/your-username/ecommerce-website.git
cd ecommerce-website
```

### Run Locally
Open `index.html` directly in your browser, or use Live Server:
```bash
# With VS Code Live Server extension
Right-click index.html → "Open with Live Server"
```

## Notes
- The sticky header and mobile menu toggle are handled in `java.js`.
- Font Awesome icons are loaded via CDN, so an internet connection is required for icons to display.
- Replace placeholder images in the `image/` folder with your own assets.
